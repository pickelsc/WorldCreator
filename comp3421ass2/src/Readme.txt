
Controls:

w: forward
s: backward
a: left
d: right

up: look up (not implemented)
down: look down (not implemented)
left: turn left
right: turn right

z: move up vertically [ground mode is off]
x: move down vertically [ground mode is odd]

c: toggle ground mode
v: toggle follow mode

[: decrease trees
]: increase tress


Extensions:

L-Systems have been added. These were generated using a variant of the Pakula-L System.
Maximum supported depth is 6 but the default display is set to 5. Growth

Complex Models was attempted. The avatar has an idle animation to make it more life like and
the Rocket Alpha is automatically placed on the highest point on the map and blasts-off
if the player approaches it
